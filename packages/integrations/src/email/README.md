# Email Integration (Resend)

## Overview
Email integration via Resend for transactional and conversational workflows.

## Key Features (Phase 3)
- Outbound email via Resend API
- Inbound email parsing via MX records + webhooks
- React Email templates
- Agent-specific email addresses (e.g., sales-worker@agents.domain.com)
