# Slack Integration

## Overview
Slack app integration via OAuth 2.0 and Event API.

## Key Features (Phase 3)
- OAuth 2.0 workspace authorization
- Event API for @mentions and DMs
- Block Kit formatted responses
- Human-in-the-loop approval buttons

## Scopes Required
- `channels:history`
- `chat:write`
- `app_mentions:read`
- `im:history`
